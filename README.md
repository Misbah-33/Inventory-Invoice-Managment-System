# 🚀 Project Overview

The Inventory & Invoice Management System is designed to help businesses manage inventory records, customer information, sales transactions, invoice generation, and reporting through a single web application.

The system eliminates the need for manual record keeping and provides a user-friendly interface for managing day-to-day business operations.

---

# ✨ Features

## 📊 Dashboard

* View total products
* View total customers
* View total invoices
* View total revenue
* View total profit
* View paid amount
* View pending amount

## 📦 Product Management

* Add new products
* Store product prices
* Support multiple currencies
* View product inventory
* Delete products

## 👥 Customer Management

* Add customer information
* Store phone numbers
* Store email addresses
* Store country information
* Manage customer records

## 💰 Sales Management

* Record sales transactions
* Select products and customers
* Calculate profit automatically
* Track payment status
* View sales history

## 🧾 Invoice Management

* Generate customer invoices
* Filter invoices by date range
* Display customer details
* Calculate invoice totals
* Download invoices as PDF

## 📈 Reports Management

* Generate sales reports
* Filter reports by customer
* Filter reports by date range
* View revenue summaries
* View profit analysis
* Export reports as PDF

# ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Misbah-33/inventory-invoice-system.git
```

### Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any modern browser.

No server or database setup is required.

# 🔄 System Workflow

1. Add products to the inventory.
2. Add customer information.
3. Record sales transactions.
4. Generate invoices for customers.
5. Download invoices in PDF format.
6. Generate sales reports.
7. Monitor performance through the dashboard.
